## 📖 Overview  
A simple Python terminal game where you try to guess a randomly selected number between **1 and 100**.  
The program gives hints whether your guess should be higher or lower until you find the correct number.

---

## 🕹️ How to Play  
1. Run the program.  
2. Enter your guess (between 1 and 100).  
3. The program will guide you with hints:  
   - `"Higher number please!!!"` if your guess is too low.  
   - `"Lower number please!!"` if your guess is too high.  
4. Keep guessing until you find the correct number.  
5. The program tells you how many attempts you made.

---

## ✨ Features  
- Random number generation between 1–100  
- Interactive hints (`Higher` / `Lower`)  
- Tracks number of attempts  
- Fun and beginner-friendly Python project
